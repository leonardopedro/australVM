### Issue 2: Austral Primitives

**Missing**: `trappingAdd`, `__union_new`, `__record_new` etc.

**Quick Fix**: Pass 3 (scan body) must capture ALL used symbols and declare dummy stubs in JIT **before** compiling bodies.

```rust
// In pre-scan loop:
for name in used_by_body {

        // Declare stub 0-argument func

    }
}
```
